# Robot

## Installation
```
npm install
```

```
npm install webpack-dev-server webpack -g
```

## Deployment
```
npm run build:dll   //提取第三方依赖，避免重复打包
npm start       //开发模式
npm run build       //生产模式
```
> 第三方依赖发生变动后需要运行一次`npm run build:dll`
